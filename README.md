날씨 정보를 알려주는 앱
==

What Used
--
- 위치 정보: 플러터에서 제공하는 ```geolocator``` 라이브러리
- 날씨 정보: [무료 API](https://openweathermap.org/]) 사용
- 이미지 정보: 무료로 제공하는 이미지 사용
