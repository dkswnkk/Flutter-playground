어려웠던 점
--

```dart
class MyButton extends StatelessWidget {
  final Widget image;
  final Widget text;
  final Color color;
  final double radius;
  final VoidCallback onPressed;

  MyButton(
      {required this.image,
      required this.text,
      required this.color,
      required this.radius,
      required this.onPressed});
 ```
- flutter가 2.0으로 업데이트 됨에 따라서 기존의 매개변수 코드를 사용할 수 없게 되었다. (구 버전의 코드를 사용했다가 많은 삽질을 했다.)
